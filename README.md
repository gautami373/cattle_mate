# 🐄 Cattle Management System

A simple **C++ console-based application** to manage cattle records. This project demonstrates the implementation of **Object-Oriented Programming (OOP)** concepts and basic **CRUD (Create, Read, Update, Delete)** operations.

## 📌 Features

* ➕ Add new cattle records
* 📋 Display all cattle records
* 🔍 Search cattle by ID
* ✏️ Update cattle information
* 🗑️ Delete cattle records
* 🚪 Exit the application

## 🛠️ Technologies Used

* C++
* Object-Oriented Programming (OOP)
* Arrays
* Functions
* Console-based Interface

## 📊 Cattle Information Stored

Each cattle record contains:

* Cattle ID
* Breed
* Category (Cow / Buffalo / Calf)
* Age
* Milk Yield (Liters per Day)
* Health Status
* Last Vaccination Date
